# Swap tokens

## Overview

This guide will help you interact with Swap, which is a fork of Uniswap v2 that Taiko has deployed only for testing purposes.

## Prerequisites

- You must have some tokens (at least ETH) on Taiko (Alpha-3 testnet). You can [receive some ETH, HORSE, and BLL from the faucets](/docs/guides/receive-tokens) and then [bridge them to Taiko](/docs/guides/use-the-bridge).
