# Differences from Ethereum

| Parameter        | Ethereum       | Taiko (alpha-3/alpha-4)      |
| ---------------- | -------------- | ---------------------------- |
| Block gas limit  | 30,000,000 gas | 6,000,000 gas                |
| Block gas target | 5,000,000 gas  | None (fixed block gas limit) |
| Block time       | 12 seconds     | Any                          |
